Detailed Description

Interface for managing physics-related operations in the application.

The IPhysic interface provides a method to apply physical forces and update entities' states based on physics calculations.

Apply forces to entities in the registry.

This method updates the state of entities in the given ECS (Entity-Component-System) registry by applying physical forces over a specified time interval.

Parameters
registryA reference to the ECS registry containing the entities and their components.
deltaTimeThe elapsed time since the last physics update, used for calculations.
