\doxysubsection{Detailed Description}
Interface for module integration. 

The \doxylink{classEngine_1_1Module_1_1IModule}{IModule} interface provides the necessary methods to implement a new module and integrate it into the application\textquotesingle{}s core. It includes functionalities for loading, unloading, and retrieving module information.
